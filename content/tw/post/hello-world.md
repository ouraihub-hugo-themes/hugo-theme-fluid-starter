歡迎使用 Hugo Theme Fluid！這是一個基於 hexo-theme-fluid 移植的 Material Design 風格部落格主題。

## 特色

- 🎨 Material Design 風格
- 🌙 深色/淺色模式切換
- 🔍 本地搜尋功能
- 📱 響應式設計
- 💬 多種評論系統支援
- 🌐 多語言支援

## 程式碼高亮

```javascript
function hello() {
  console.log('Hello, Fluid!');
}

hello();
```
