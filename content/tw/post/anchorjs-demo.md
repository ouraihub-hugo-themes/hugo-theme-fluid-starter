本文演示 Hugo Theme Fluid 的錨點連結功能。錨點連結可以讓你方便地分享文章的特定章節。

<!--more-->

## 什麼是錨點連結

錨點連結（Anchor Links）是一種允許使用者直接跳轉到頁面特定位置的連結。當你將滑鼠懸停在標題上時，會看到一個錨點圖示（預設是 `#` 或 `§`），點擊它可以：

1. 更新瀏覽器網址列的 URL
2. 複製連結分享給他人
3. 他人開啟連結後會自動捲動到對應位置

## 功能特性

### 支援多級標題

錨點連結支援 h1 到 h6 的所有標題級別。預設配置會為所有標題新增錨點。

### 靈活的位置配置

錨點圖示可以配置在標題的左側或右側：

- **左側放置**：錨點圖示顯示在標題文字的左邊
- **右側放置**：錨點圖示顯示在標題文字的右邊

### 可配置的顯示方式

錨點圖示的顯示方式有兩種：

- **hover**：僅在滑鼠懸停時顯示（預設）
- **always**：始終顯示

### 自訂圖示

你可以自訂錨點圖示，常用的選項包括：

- `§`（預設）
- `#`
- `❡`
- `🔗`

## 配置說明

在 `params.toml` 中配置錨點連結：

```toml
[fun_features.anchorjs]
enable = true                    # 是否啟用
element = "h1,h2,h3,h4,h5,h6"   # 要新增錨點的元素
placement = "left"               # 位置：left 或 right
visible = "hover"                # 顯示方式：hover 或 always
icon = ""                        # 自訂圖示，留空使用預設
```

## 使用場景

### 技術文件

在編寫技術文件時，錨點連結非常有用。讀者可以直接分享某個 API 或配置項的連結。

### 教學文章

教學文章通常有多個步驟，錨點連結可以讓讀者快速跳轉到特定步驟。

### FAQ 頁面

FAQ 頁面的每個問題都可以有獨立的錨點連結，方便使用者分享特定問題的答案。

## 與其他功能的配合

### 與 TOC 目錄配合

錨點連結與文章目錄（TOC）完美配合。點擊 TOC 中的標題會跳轉到對應位置，同時更新 URL 中的錨點。

### 與搜尋功能配合

搜尋結果可以直接連結到文章的特定章節，提升使用者體驗。

## 總結

錨點連結是一個簡單但實用的功能，它可以：

- 提升文章的可分享性
- 改善長文章的導覽體驗
- 與 TOC、搜尋等功能協同工作
