# 主題特性

Hugo Fluid 是一個功能豐富的 Material Design Hugo 主題，採用現代化前端技術。

## 核心特性

### 🎨 Material Design
- 簡潔現代的 Material Design 介面
- 響應式設計適配所有裝置
- 流暢的動畫和過渡效果
- 卡片式佈局系統

### 🌙 深色模式支援
- 自動系統主題檢測
- 手動主題切換（自動/淺色/深色）
- 持久化主題偏好
- 平滑主題過渡

### 📱 響應式設計
- 行動優先的設計方法
- 針對所有螢幕尺寸最佳化
- 觸控友好的導航
- 自適應佈局

## 頁面類型

### 🏠 首頁
- 視差效果的 Hero Banner
- 副標題打字機效果
- 精選文章展示
- 可自訂標語和描述

### 📝 文章頁面
- 目錄（TOC）側邊欄
- 閱讀進度指示器
- 程式碼語法高亮
- 一鍵複製程式碼
- 社群分享按鈕
- 版權聲明

### 📚 歸檔頁面
- 時間軸式歸檔佈局
- 分類和標籤組織
- 搜尋功能
- 分頁支援

### 🔗 特殊頁面
- 個人資訊關於頁面
- 友情連結頁面
- 自訂 404 錯誤頁面
- 搜尋結果頁面

## 互動功能

### 🔍 搜尋功能
- Pagefind 驅動的快速本地搜尋
- 多語言搜尋支援
- 即時搜尋建議
- 高亮搜尋結果

### 📖 閱讀體驗
- 平滑滾動的目錄
- 閱讀進度條
- 回到頂部按鈕
- 圖片懶載入
- 點擊圖片放大

### 💬 評論系統
支援多種評論系統：
- **Giscus**（GitHub Discussions）
- **Utterances**（GitHub Issues）
- **Disqus**
- **Gitalk**
- **Waline**
- **Valine**

## 技術特性

### ⚡ 效能
- Hugo 靜態網站生成
- 最佳化的 CSS 和 JavaScript
- 圖片懶載入
- 最小化資源載入

### 🛠 現代技術棧
- **TypeScript** 提供型別安全的 JavaScript
- **Tailwind CSS** 實用優先的樣式
- **PostCSS** CSS 處理
- **esbuild** 快速打包

### 🌍 多語言支援
內建支援 9 種語言：
- 简体中文（簡體中文）
- 繁體中文（Traditional Chinese）
- English（英語）
- 日本語（日語）
- 한국어（韓語）
- Español（西班牙語）
- Deutsch（德語）
- Français（法語）
- Português（葡萄牙語）

### 📊 統計分析整合
- Google Analytics
- 百度統計
- Umami Analytics
- 不蒜子訪客計數器

## 內容特性

### 📝 豐富內容支援
- 擴展語法的 Markdown
- 數學公式（MathJax/KaTeX）
- Mermaid 圖表
- 多主題程式碼高亮
- 增強內容的短代碼

### 🏷 組織方式
- 階層式分類支援
- 標籤雲
- 按日期歸檔
- 自訂分類法

### 🖼 媒體支援
- 圖片畫廊
- 影片嵌入
- 音訊播放器
- 所有媒體懶載入

## 自訂化

### 🎨 樣式
- CSS 自訂屬性主題化
- 可自訂色彩方案
- 字型選項
- 佈局配置

### ⚙ 配置
- 豐富的主題參數
- 每頁配置
- 選單自訂
- 外掛系統

### 🔧 開發
- 開發時熱重載
- TypeScript 支援

- 自動化測試

## SEO 和社群

### 🔍 SEO 最佳化

### 📱 社群功能
- 社群分享按鈕

- 社群 Meta 標籤
