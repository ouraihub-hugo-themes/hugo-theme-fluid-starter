本文演示 Hugo Theme Fluid 的锚点链接功能。锚点链接可以让你方便地分享文章的特定章节。

<!--more-->

## 什么是锚点链接

锚点链接（Anchor Links）是一种允许用户直接跳转到页面特定位置的链接。当你将鼠标悬停在标题上时，会看到一个锚点图标（默认是 `#` 或 `§`），点击它可以：

1. 更新浏览器地址栏的 URL
2. 复制链接分享给他人
3. 他人打开链接后会自动滚动到对应位置

## 功能特性

### 支持多级标题

锚点链接支持 h1 到 h6 的所有标题级别。默认配置会为所有标题添加锚点。

### 灵活的位置配置

锚点图标可以配置在标题的左侧或右侧：

- **左侧放置**：锚点图标显示在标题文字的左边
- **右侧放置**：锚点图标显示在标题文字的右边

### 可配置的显示方式

锚点图标的显示方式有两种：

- **hover**：仅在鼠标悬停时显示（默认）
- **always**：始终显示

### 自定义图标

你可以自定义锚点图标，常用的选项包括：

- `§`（默认）
- `#`
- `❡`
- `🔗`

## 配置说明

在 `params.toml` 中配置锚点链接：

```toml
[fun_features.anchorjs]
enable = true                    # 是否启用
element = "h1,h2,h3,h4,h5,h6"   # 要添加锚点的元素
placement = "left"               # 位置：left 或 right
visible = "hover"                # 显示方式：hover 或 always
icon = ""                        # 自定义图标，留空使用默认
```

## 使用场景

### 技术文档

在编写技术文档时，锚点链接非常有用。读者可以直接分享某个 API 或配置项的链接。

### 教程文章

教程文章通常有多个步骤，锚点链接可以让读者快速跳转到特定步骤。

### FAQ 页面

FAQ 页面的每个问题都可以有独立的锚点链接，方便用户分享特定问题的答案。

## 与其他功能的配合

### 与 TOC 目录配合

锚点链接与文章目录（TOC）完美配合。点击 TOC 中的标题会跳转到对应位置，同时更新 URL 中的锚点。

### 与搜索功能配合

搜索结果可以直接链接到文章的特定章节，提升用户体验。

## 总结

锚点链接是一个简单但实用的功能，它可以：

- 提升文章的可分享性
- 改善长文章的导航体验
- 与 TOC、搜索等功能协同工作
