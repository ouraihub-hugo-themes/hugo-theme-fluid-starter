欢迎使用 Hugo Theme Fluid！这是一个基于 hexo-theme-fluid 移植的 Material Design 风格博客主题。

## 特性

- 🎨 Material Design 风格
- 🌙 深色/浅色模式切换
- 🔍 本地搜索功能
- 📱 响应式设计
- 💬 多种评论系统支持
- 🌐 多语言支持

## 代码高亮

```javascript
function hello() {
  console.log('Hello, Fluid!');
}

hello();
```
